# Image Recognition Task

## Objective
Train an image recognition model using Google Teachable Machine and classify an input image using Python.

## Classes
- Phone
- Pen

## Steps
1. Created an image classification project using Google Teachable Machine.
2. Trained the model using two classes: Phone and Pen.
3. Exported the model as TensorFlow → Keras.
4. Used Google Colab to load the model.
5. Loaded an input image (Test.jpeg).
6. Predicted the image class using the trained model.
7. Saved a screenshot of the prediction output.

## Files
- keras_model.h5
- labels.txt
- Python script.py
- Test.jpeg
- Screenshot of output.jpeg
